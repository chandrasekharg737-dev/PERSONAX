# ✨ PERSONAX - Personality Meets the Skin

Personax is a modern, premium AI-powered personality and skincare assistant. It analyzes your unique vibe and skin profile to curate a holistic beauty and style journey.

## 🚀 Deployment to Render

This project is configured as a Node.js Web Service for maximum reliability.

1. **Push to GitHub**: Upload all files (except those in `.gitignore`) to a new repository.
2. **Connect to Render**:
   - Create a new **Web Service**.
   - Connect your GitHub repository.
   - **Build Command**: `npm install`
   - **Start Command**: `npm start`
3. **Environment Variables**:
   - Go to the **Environment** tab in Render.
   - Add a secret file called `.env` or add your variables (like `PORT`) directly.

## 🎨 Features
- **Dynamic Quiz**: 6-step personalized analysis.
- **Vibrant UI**: Glassmorphic design with moving gradients.
